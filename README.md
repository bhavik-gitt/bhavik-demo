# 🚀 Bhavik Ratnottar — Personal Portfolio

A modern, responsive **personal portfolio website** built with HTML and CSS, showcasing my skills, projects, education, experience, certificates, achievements, and more.

## 📌 About the Project

This project is a single-page portfolio website featuring:

- Clean, responsive design that works on all screen sizes
- **Dark / Light mode** toggle (remembers your preference via `localStorage`)
- Smooth scroll navigation with an active-link highlight on scroll
- Sections: About, Skills, Projects, Education, Experience, Certificates & Workshops, Achievements, Resources, Contact

## 📂 Project Structure

```
bhavik-demo/
│
├── index.html   ← Main portfolio page (all sections & JavaScript)
├── style.css    ← All styles, CSS variables, dark mode & responsive rules
└── README.md    ← Project documentation
```

## ✨ Features

- **Responsive** — mobile-first layout using CSS Grid and Flexbox
- **Dark Mode** — toggled via a button; persisted in `localStorage`
- **Scroll-spy** — active nav link updates as you scroll through sections
- **Hover animations** — cards lift with shadow on hover
- **Accessible** — semantic HTML, `aria-label` on interactive elements
