## 📌 About the Project

<table>
<tr>
<td>

A fully responsive, **dark-mode-only** personal portfolio built entirely with **HTML5**, **CSS3**, and **Vanilla JavaScript** — no frameworks, no build tools, no bloat.

Every section is crafted to be clean, accessible, and visually engaging:

- 🌌 **Canvas starfield** hero with animated shooting stars
- 🔠 **Letter-by-letter** name animation + typing role effect
- 🃏 **3D card tilt** on project cards (mouse-tracked perspective)
- 🔍 **Scroll-spy** navigation with smooth active-state transitions
- ♿ **ARIA-labelled** semantic landmarks throughout
- 🔎 **Full SEO suite** — OG, Twitter Card, JSON-LD, sitemap

</td>
</tr>
</table>

## ✨ Features

<div align="center">

| Feature | Description | Status |
|:---:|:---|:---:|
| 🌙 | **Dark Mode Only** — `data-theme="dark"` enforced globally; eye-friendly colour palette | ✅ |
| 📱 | **Fully Responsive** — Mobile-first CSS Grid & Flexbox; hamburger nav at ≤ 1050 px | ✅ |
| 🌌 | **Canvas Starfield** — Animated 2D starfield + shooting stars in the hero background | ✅ |
| ⌨️ | **Typing Animation** — Role text cycles through titles with blinking cursor | ✅ |
| 🃏 | **3D Card Tilt** — Mouse-tracked `perspective` + `rotateX/Y` on project cards | ✅ |
| 🔮 | **Cursor Glow** — Smooth lerp-interpolated glow follows the cursor | ✅ |
| 🔍 | **Scroll-Spy Nav** — Active link updates automatically as sections enter viewport | ✅ |
| 🎨 | **Categorised Skills** — Skill groups with per-category accent colours + tech icons | ✅ |
| ♿ | **Accessible** — Semantic HTML5 landmarks, `aria-label` on all interactive elements | ✅ |
| 💫 | **Scroll-Reveal** — Elements fade + slide in via `IntersectionObserver` | ✅ |
| ⚡ | **Zero Build Step** — Open `index.html` in any browser; no npm, no bundler | ✅ |
| 🔎 | **SEO Ready** — OG, Twitter Card, JSON-LD structured data, canonical, sitemap | ✅ |
| 🎯 | **Reduced-Motion** — All animations respect `prefers-reduced-motion` | ✅ |

## ⚡ Performance & Accessibility

<table>
<tr>
<td width="50%">

**🚀 Performance**
- Zero npm dependencies — clone & open in browser
- Inline CSS custom properties — instant theme consistency
- Canvas-based starfield — GPU-accelerated 2D rendering
- `IntersectionObserver` scroll-reveal — no scroll event polling
- `requestAnimationFrame` cursor glow — smooth 60 fps lerp

</td>
<td width="50%">

**♿ Accessibility & SEO**
- Semantic landmarks: `<header>` `<main>` `<section>` `<footer>`
- `aria-label` on every button, link, and interactive element
- `prefers-reduced-motion` — all animations disabled gracefully
- Full SEO: OG, Twitter Card, JSON-LD Person + WebSite schema
- `robots.txt` · `sitemap.xml` · canonical URL · meta description

</td>
</tr>
</table>
